## ⚙️ Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/ultraxcode-com/ultra-x-code
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Lanza el proyecto en local:
   ```bash
   npm run dev
   ```
